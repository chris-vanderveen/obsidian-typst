# Shortcuts

After selecting text inside a formula, press one of the keyboard shortcuts below to generate a shortcut. The selected text will be placed into `早`.

This feature corresponds to the "Visual snippets" feature in [Obsidian LaTeX Suite](https://www.obsidianstats.com/plugins/obsidian-latex-suite).

---

The following shortcuts use lowercase letters, numbers, or symbols.

## [Cancel](https://typst.app/docs/reference/math/cancel/)

- <kbd>!</kbd>: `cancel(早)`

## [Left/Right](https://typst.app/docs/reference/math/lr)

- <kbd>l</kbd>: `lr(早)`
- <kbd>a</kbd>: `abs(早)`
- <kbd>n</kbd>: `norm(早)`
- <kbd>f</kbd>: `floor(早)`
- <kbd>c</kbd>: `ceil(早)`
- <kbd>e</kbd>: `round(早)`

## [Roots](https://typst.app/docs/reference/math/roots)

- <kbd>2</kbd>: `sqrt(早)`
- <kbd>3</kbd>: `root(3, 早)`
- <kbd>4</kbd>: `root(4, 早)`
- <kbd>5</kbd>: `root(5, 早)`
- <kbd>6</kbd>: `root(6, 早)`
- <kbd>7</kbd>: `root(7, 早)`
- <kbd>8</kbd>: `root(8, 早)`
- <kbd>9</kbd>: `root(9, 早)`

## [Stretch](https://typst.app/docs/reference/math/stretch)

- <kbd>^</kbd>: `stretch(早)^"|"`
- <kbd>_</kbd>: `stretch(早)_"|"`

## [Styles](https://typst.app/docs/reference/math/styles)

- <kbd>u</kbd>: `upright(早)`
- <kbd>b</kbd>: `bold(早)`

---

Below, uppercase letters are used except for `Variants/sans`.

## [Under/Over](https://typst.app/docs/reference/math/underover)

- <kbd>U</kbd>: `underbrace(早, "|")`
- <kbd>O</kbd>: `overbrace(早, "|")`

## [Variants](https://typst.app/docs/reference/math/variants)

- <kbd>s</kbd>: `sans(早)`
- <kbd>F</kbd>: `frak(早)`
- <kbd>M</kbd>: `mono(早)`
- <kbd>B</kbd>: `bb(早)`
- <kbd>C</kbd>: `cal(早)`
- <kbd>S</kbd>: `scr(早)`
